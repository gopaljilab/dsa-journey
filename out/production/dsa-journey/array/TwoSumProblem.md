## ⚡ Algorithm
- Iterate over each element with two nested loops.
- Check every pair `(i, j)` where `i != j`.
- If `nums[i] + nums[j] == target`, return `[i, j]`.

---

## ⏱️ Time Complexity
- **Brute-force approach:** O(n²)
- **Space Complexity:** O(1) (No extra data structures used)
